Elasticsearch

Рекомендательная система локаций для бизнеса: Go + Elasticsearch

Недавно я завершил работу над проектом go_es_analytical_system — рекомендательной системой для поиска и анализа локаций для различных типов бизнеса. Хочу поделиться опытом и рассказать, какие задачи решает эта система и какие технологии использовались. Зачем это нужно? Представьте ситуацию: вы хотите открыть кафе, ресторан или магазин. Где лучше разместить бизнес? Какие факторы учесть? Трафик — сколько людей проходит мимо локации Конкуренция — сколько похожих заведений уже есть рядом Демография — кто живёт и работает в этом районе Доступность — удобно ли добираться Ручной анализ таких данных занимает недели. Моя система делает это за секунды, используя комбинацию PostgreSQL для справочников и Elasticsearch для полнотекстового поиска и ранжирования. ...
